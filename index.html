What is Mobile Responsive Web Design?

People view websites from all kinds of devices. In the CodeHS Web Design course, we are learning to make websites that are usable from all devices using mobile responsive web design.


Mobile Responsive Web Design

An important part of web design is making sure that people can see and use your website no matter what device they are viewing it from. Whether it's a desktop, laptop, tablet, or phone, we are learning how to build websites in this course that look great from any device!

The idea is to make your site responsive. Using Bootstrap, we are learning how to effectively build sites that stretch and collapse to fit screens of any size. Just watch what happens when we resize the window on this site!


Try it yourself! Resize this window and see how the site responds.

Try viewing this site from your phone as well! These are the kinds of websites students will be able to create at the end of this course.
